# E-Commerce-Shopping-Cart
A simple e-commerce product catalog app where users can browse products, add items to a cart, and view the total price dynamically using modern web technologies.
This project is a simplified E-commerce Product Catalog application that allows users to browse products, view details, add items to a shopping cart, and calculate the total price dynamically. It demonstrates the core functionality of an online shopping system using modern web technologies.
Users can explore a list of products rendered dynamically, manage their cart by adding or removing items, and see real-time updates to the total cost. The project focuses on creating a smooth and interactive user experience while practicing frontend development concepts.
